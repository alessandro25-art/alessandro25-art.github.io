## US-001 (Fix: cloni del carousel non rispondono ai click)
- Replaced per-element `querySelectorAll('.card-toggle').forEach` and separate `.card-thumb` forEach with a single delegated listener on `.work-carousel`
- Delegation handles both `.card-toggle` and `.card-thumb` clicks (thumb resolves to its card's toggle button)
- On each toggle, computes twin card index (`origCount = allCards.length / 2`, twinIndex flips between first and second half) and silently syncs hidden/is-open/aria-expanded/text/arrow on the twin
- `window.cardOpen` now computed from actual DOM state: `any .card-toggle[aria-expanded=true]` across all cards
- Files changed: index.html
- Learnings:
  - Delegation must be on a container that exists before cloning — the carousel element is created in HTML, cloning happens in script block 2, so delegating on `.work-carousel` in script block 1 is safe
  - `var tb` declared in both if/else branches is valid non-strict JS (hoisted once, only one branch executes)
  - `window.cardOpen = false` in carousel IIFE (script block 2) only runs at page load, so it doesn't interfere with runtime delegation management

## US-001 (Fix: infinite carousel loop stops at the end instead of looping)
- Replaced `var halfWidth = carousel.scrollWidth / 2` with `origCards2[origCards2.length / 2].offsetLeft` to get the exact reset threshold
- Updated the RAF step reset check to use `loopWidth` instead of `halfWidth`
- Files changed: index.html
- Learnings:
  - `scrollWidth / 2` is inaccurate because scrollWidth includes padding on both ends; the first clone's `offsetLeft` is the exact pixel where the second set starts
  - Must query cards AFTER cloning so `querySelectorAll('.card')` returns all cards (originals + clones); `length / 2` indexes the first clone

## US-002 (Fix: clicking a card should snap it to center of the carousel viewport)
- Replaced `card.scrollIntoView(...)` with `workCarousel.scrollTo({ left: target, behavior: 'smooth' })` where `target = card.offsetLeft - workCarousel.offsetLeft - (workCarousel.clientWidth / 2 - card.offsetWidth / 2)`
- Files changed: index.html
- Learnings:
  - `scrollIntoView` scrolls the nearest scrollable ancestor which can be the page, not the carousel — manual `scrollTo` on the container is reliable
  - In script block 1, use `workCarousel` (in scope) not `carousel` (only in scope in script block 2)
