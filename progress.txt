## US-001 (Pausa carousel quando si apre un Read more)
- Added `window.cardOpen` boolean shared between the card-toggle script and the carousel IIFE
- Card toggle sets `window.cardOpen = true` on expand, `false` on collapse
- Carousel RAF loop checks `!window.cardOpen` in addition to `!paused`
- Files changed: index.html
- Learnings:
  - Card toggle and carousel are in separate `<script>` blocks — local variables aren't shared; use `window.*` for cross-script state
---

## US-001 (Click sull'immagine apre Read more e ferma il carousel)
- Added `cursor: pointer` to `.card-thumb` CSS rule
- Added click listeners on `.card-thumb` that call `btn.click()` on the sibling `.card-toggle`, reusing all expand/collapse and `window.cardOpen` logic
- Files changed: index.html
- Learnings:
  - `thumb.closest('.card').querySelector('.card-toggle')` is the cleanest way to find the toggle from within the same card
  - Delegating to `btn.click()` avoids duplicating the expand/pause logic
