# Portfolio — Alessandro Demichelis

## Purpose
Career fair portfolio page. Single file (index.html) inspired by tanmade.com.
Hosted on GitHub Pages at https://alessandrodemichelis.github.io

## File structure
- index.html — main portfolio page (self-contained HTML/CSS/JS)
- CV/CV_Alessandro_Demi.pdf — CV download target
- Lavori/ — 3 project PDFs linked from project cards

## Design reference
https://www.tanmade.com/ — minimal, editorial, typographic. No templates, no gradients.
White background (#FAFAFA), Playfair Display 700 for H1, DM Sans 300/400/500 for everything else.
Max-width 720px, padding 24px, generous whitespace between sections.

## Writing style
All copy must use the Humanizer skill (/humanizer).
Alessandro's voice: direct, honest, concise. No AI puffery, no significance inflation.
Voice profile is defined in ~/.claude/commands/humanizer.md
Do NOT use: "passionate about", "driven professional", "merit scholarship recipient",
"significant contribution", "pivotal", "testament", or any AI vocabulary words.

## QR code
Points to https://alessandrodemichelis.github.io (update before deploy if username differs)
Generated client-side via qrcode.js CDN — no install needed.

## Deploy
GitHub Pages: repo name must be exactly `alessandrodemichelis.github.io`
Upload index.html + CV/ + Lavori/ folders via GitHub Desktop.
Settings > Pages > main branch / root > Save. Live in ~2 min.
